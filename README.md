# NER Annotation — Nigerian News2 Dataset

## Overview
A Named Entity Recognition annotation dataset of 20 sentences
collected from Nigerian news sources including Punch, Vanguard,
and BusinessDay.

## Labels
- ORGANISATION — Companies and businesses
- PERSON — Named individuals
- LOCATION — Cities, states, countries
- DATE — All date references
- MONEY — Currency amounts
- PERCENTAGE — Percentage figures
- GOVERNMENT_AGENCY — Government bodies and agencies

## Tools Used
- Doccano (annotation)
- JSONL format (export)

## Source
Real sentences collected from Nigerian news websites
March 2024

## Purpose
Portfolio project demonstrating NER annotation skills
using industry standard tools and formats.# ner-annotation-nigerian-news-2
Named Entity Recognition (NER) project for annotating Nigerian news articles.
